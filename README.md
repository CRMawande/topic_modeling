# Topic Modeling Project (DLBDSEDA02) 
NLP analysis of NYC 311 complaints using LDA/NMF. 
## Setup 
1. Create venv: python -m venv venv 
2. Activate: venv\Scripts\activate.bat 
3. Install deps: pip install -r requirements.txt 
4. Download NYC 311 data to data\raw\ from https://data.cityofnewyork.us/Social-Services/311-Service-Requests-from-2010-to-Present/erm2-nwe9 
5. Run notebooks in order: 01 -> 02 -> 03.
